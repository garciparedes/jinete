storers package¶

Entities whose responsibility is to deliver problem solutions.

jinete.storers.formatters

The set of definitions whose aim is to format the resulting artifacts.

jinete.storers.plots

Plotting storers module to define the set of storers whose output is a graphical artifact.

jinete.storers.abc

The set of abstract definitions for the storers module.

jinete.storers.file

The set of definitions to store results in filesystems.

jinete.storers.naive

The set of definitions to not to store results.

jinete.storers.prompt

The set of definitions to store results as a prompt message.

jinete.storers.sets

The set of definitions to use more than one storer at the same time.
